Docker

OpenMPF is installed using the Docker container platform.

To use prebuilt Docker images, refer to the "Quick Start" section of the documentation for the OpenMPF Workflow Manager image on DockerHub.

For more information, including how to setup Docker, and build and deploy OpenMPF Docker images, refer to the openmpf-docker README.

Additionally, if you would like to install OpenMPF across multiple physical or virtual machines, then refer to the openmpf-docker Swarm Deployment Guide.

Please review the page on License and Distribution.
